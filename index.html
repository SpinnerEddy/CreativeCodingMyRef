学ぶメリット

ランダム

  • 色、動きの速度から描画する物体の形の決定まであらゆる要素で使える。
  • シード値があり、これを設定することである程度コントロールもできる。
  • これを使用するだけである程度出力にランダム性が出て、実行する毎に違った結果が得られるようになる。(ただし、乱用するとコントロールが難しくなるので注意)

ノイズ

  • ランダムと同様に動きの速度や方向、色などあらゆる要素に使える。
  • ランダムと比べると、滑らかなランダム要素を持たせられる。
  • シード値があり、これを設定することである程度コントロールもできる。

線形補間

  • 任意の点と点を線で結ぶ、基本的な考え方の一つ(一次関数)
  • イージングに繋がる。
  • 色、位置などあらゆる考え方に使える。

イージング

  • 任意の値1から値2までの変化を式で表現するもの
  • 式は様々な種類がある。
  • これも色、位置などあらゆる考え方に使える。
  • 点Aから点Bまでを最初は加速、最後は減速させて到達させる とかが可能

三角関数

  • 物体の向きを知る上では必須。ここから自在に向きを付けることもできる。
  • 幾何学図形を作ったり、波形っぽいものを作ったりなどもできる。
  • 極座標という考えを理解する上では重要。極座標は(原点からの距離, 角度)で表す。

ベクトル

  • 方向と長さを一つとして扱う概念。
  • 任意の物体の方向を知る、物体を任意の方向に動かす、2点間距離を測るなどを出来ることがとても多い。
  • 力(自然の物理法則に沿った動き)を身につけるには必要

ベクトル

ここでは下記について説明する。

  1. ベクトルの基本的な知識
  2. p5.jsで扱えるベクトルの関数と使い所

ベクトルとは

  • 方向と長さを持つもの
  • 位置、物体を動かす方向、物体間の距離など様々なものを表すのに使用することができるとても汎用的な知識

ベクトルの関数と使い所(今回は2次元のみ)

加算

let a = createVector(3,5);  // 原点から(3,5)を結ぶベクトルを作る
let b = createVector(2,4);  // 原点から(2,4)を結ぶベクトルを作る

let c = p5.Vector.add(a, b);  // c = a + b = (3+2, 5+4) = (5, 9)
a.add(b);                     // aにbを足し、a = (5, 9)になる (aの値が変わることに注意)

サンプルはこちら

  1. 減算
let a = createVector(3,5);  // 原点から(3,5)を結ぶベクトルを作る
let b = createVector(2,4);  // 原点から(2,4)を結ぶベクトルを作る

let c = p5.Vector.sub(a, b);  // c = a - b = (3-2, 5-4) = (1, 1)
a.sub(b);                     // aからbを引き、a = (1, 1)になる (aの値が変わることに注意)

  1. 掛け算

a.div(b);                     // aからbを引き、a = (2, 1)になる (aの値が変わることに注意)
